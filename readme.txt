-------------------------Bienvenue sur  ZX Notes-------------------------------------------
-------------------------------------------------------------------------------------------

ZX Note est un éditeur de texte qui vous permet de chiffrer vos écrits avec 
méthode SHA256, ce qui garantit la sécurité de vos fichiers.

La clé privé est indispensable pour chiffrer / déchiffrer ,vous pouvez l'inventer,
vous devrez utiliser exactement la même clé privé pour chiffrer / déchiffrer.

1 - Pour lancer le chiffrement, insérez la clé privée  en haut à droite puis 
cliquez sur "chiffrer et sauvegarder"

2 - Pour déchiffrer un fichier insérez la clé privée ayant servie à chiffrer
le fichier en haut à droite de l'écran, puis cliquez sur "déchiffrer un fichier"

ZX Note is a text editor that allows you to encrypt your writings with
SHA256 method, which guarantees the security of your files.

The private key is essential to encrypt / decrypt, you can invent it,
you will need to use the exact same private key to encrypt / decrypt.

1 - To start the encryption, insert the private key at the top right then
click on "encrypt and save"

2 - To decrypt a file insert the private key used to encrypt
the file at the top right of the screen, then click on "decrypt a file"
